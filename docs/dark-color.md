# SplitWiser Option A — Color Palette

## Theme: Dark Fintech

### Base Colors
| Token | Hex | Usage |
|-------|-----|-------|
| `--bg` | `#0B0F1A` | Page background |
| `--surface` | `#131825` | Cards, sidebar, panels |
| `--surface-hover` | `#1A2035` | Hover state for surfaces |
| `--border` | `#1E2640` | Borders, dividers |

### Text Colors
| Token | Hex | Usage |
|-------|-----|-------|
| `--text` | `#E8ECF4` | Primary text |
| `--text-muted` | `#6B7A99` | Secondary text, labels, metadata |

### Accent Colors
| Token | Hex | Usage |
|-------|-----|-------|
| `--accent` | `#3ECF8E` | Primary accent, positive balances, CTAs |
| `--accent-dim` | `rgba(62,207,142,0.12)` | Accent backgrounds, active nav items |
| `--red` | `#F87171` | Negative balances, amounts owed |
| `--red-dim` | `rgba(248,113,113,0.12)` | Red icon backgrounds |
| `--amber` | `#FBBF24` | Warning, grocery/food icons |
| `--amber-dim` | `rgba(251,191,36,0.12)` | Amber icon backgrounds |
| `--blue` | `#60A5FA` | Secondary accent, utility icons |
